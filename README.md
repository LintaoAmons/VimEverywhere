# VimEverywhere
**Learn vim once, Use everywhere in every places.**

- 打造以 Vim 为基础的全键盘工作流.
	- 在 IDEA, VScode, terminal 使用 Vim，写代码
	- 在 Chrome, Firefox 中使用 Vim, 浏览网页，切换Tab，复制内容
	- 在 Joplin 使用Vim，记笔记，构建知识库
	- 使用键盘切换软件窗口，分屏，调出常用软件
